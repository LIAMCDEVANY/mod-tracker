# Mod Tracker

Mod Tracker is a full-stack MEN (MongoDB, Express, Node.js) CRUD application designed to help users track video game mods they are actively working on. Authenticated users can create, view, update, and delete mod entries, along with notes, status, and related links.

This project demonstrates full CRUD functionality, user authentication, session management, and clean MVC organization.

---

## Features

- User authentication (sign up, log in, log out)
- Create mod entries
- View a list of mods
- View detailed information for each mod
- Edit mod details (name, status, notes, links)
- Delete mods
- Persistent sessions using MongoDB
